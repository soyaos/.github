# SoyaOS

> **An Agent Operating System.** One binary, six editions, three node roles (Planet / Moon / Comet) — unifying compute and capabilities across the public internet, your intranet, and ephemeral sandboxes.

SoyaOS is named after the humble soybean: one bean, many forms (edamame, tofu, soy milk, yuba). SoyaOS is the same idea for Agents — **one kernel, many shapes**.

## Status

**Pre-release.** APIs are unstable. The v0.1.0 milestone is locked to four flagship user stories: **Compo · NewsBeam · EstateMuse · SilentCut**. When all four run end-to-end across the six editions, v0.1.0 ships.
